# Vibe Neon Runner

**Vibe Neon Runner** är ett enkelt, retroinspirerat ("neon"-stil) webbläsarspel där du styr en lysande kub och undviker inkommande hinder. Spelet ökar i svårighetsgrad och hastighet ju fler hinder du lyckas hoppa över.

## Hur man spelar

- Öppna `index.html` i din webbläsare.
- Klicka på **STARTA SPEL** eller tryck på **Mellanslag** för att börja spelet.
- Använd **Mellanslag**, **Uppåtpil**, eller **klicka med musen/pekskärmen** för att hoppa.
- Undvik de rosa hindren för att samla poäng. Hastigheten ökar för varje 300:e poäng!

## Funktioner

- Oändlig "Runner"-mekanik.
- Neonfärgad estetik med retro-rutnät ("grid") i bakgrunden.
- Partikeleffekter (hopp- och explosionsanimationer).
- Responsiv layout som anpassar sig efter webbläsarens fönsterstorlek.
- Skrivet i Vanilla HTML, CSS och JavaScript, vilket innebär att inga externa bibliotek eller ramverk krävs.

## Teknologi

- **HTML5:** Struktur för spelet och Canvas för rendering.
- **CSS3:** Stil, typsnitt, layout och "glow"-effekter.
- **JavaScript (ES6):** Spellogik, rendering i Canvas, fysik (gravitation och hopp), och partikelsystem.
